作品设计思路：
在长安历经的十三个朝代中，我们小组经过讨论觉得自己都对唐朝时期的长安比较感兴趣，所以我们决定从历史，著名人物，风俗趣事这几个方面来介绍我们心目中的长安。我们的页面主要分为四个部分，一个主页，剩下三个分别关于历史，人物，风俗趣事的页面。这个网页整体采用了轮播图的方式来展现。

每个页面的介绍：

主页：主页我们主要大致介绍了一下长安的历史，然后分别从导航栏跳转到不同的介绍页面。

源远流长页面：这个部分主要介绍在唐朝时期，中国社会经济处于上升阶段，文化先进，是历史上中国向周边国家文化与技术的一个大输出时期，兼容并蓄的社会风气，也给五胡十六国以来进居塞内的各个民族提供一个空前的交流融合环境，在过程中亦从外族文明汲取诸多。唐朝的后半叶，处于中国历史的转型期，土地、盐铁、赋税制度的改革标志着社会的缓慢变化，藩镇割据势力的壮大与周边诸民族国家的形成，对此后近千年的中国历史产生深远的影响。
该页面以唐朝重大时期为时间线，依次是唐初盛世，移都洛阳，高宗时期，五周代唐，开元之治，由盛转衰几个时期。
依次介绍了皇帝，以及皇帝做出的重大决策。这个页面点击相关的图片，会使图片放大呈现出来。

风俗趣事页面：在这个页面，我们主要通过唐朝时期长安的一些有趣的故事或者风俗来展现唐朝时期的长安。这个页面包含诗人趣事、君臣趣事、女帝禁屠、大唐服饰、以胖为美、婚姻习俗这几个板块来介绍唐朝时期的风俗趣事。

风流人物页面：在这个页面我们选取了感兴趣的诗仙李白和一代美人杨玉环这两个著名人物来展现长安风采。其中李白的介绍页面，点击那首诗，诗的内容会依次呈现出来，使整个页面活起来。
